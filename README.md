# Movie Recommendation System

## Description
A Movie Recommendation System that suggests movies based on user ratings and movie content. Uses techniques like collaborative filtering and neural networks. Built with Pandas, NumPy, Scikit-learn, TensorFlow, and Matplotlib. Data from the MovieLens dataset.

## Skills Demonstrated
- Collaborative Filtering
- Content-Based Filtering
- Evaluation Metrics

## Libraries Used
- Pandas
- NumPy
- Scikit-learn
- TensorFlow
- Matplotlib

## Dataset
- MovieLens (http://files.grouplens.org/datasets/movielens/ml-latest-small.zip)

## Usage
1. **Clone the repository:**
    ```bash
    git clone https://github.com/yourusername/movierecsystem.git
    cd movierecsystem
    ```

2. **Install the required libraries:**
    ```bash
    pip install pandas numpy matplotlib seaborn scikit-learn tensorflow requests
    ```

3. **Run the script:**
    ```bash
    python movierecsystem.py
    ```
